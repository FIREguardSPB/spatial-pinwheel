Этот архив — overlay-патч поверх текущей рабочей версии проекта у заказчика.

Что внутри:
- срочный фикс live-обновления графика через frontend polling + heartbeat publish;
- базовый контур ручных ордеров (`/api/v1/orders/market`, `/api/v1/orders/limit`) и панель в Dashboard;
- явные AI-маркеры для сигналов/сделок;
- systemd unit-файлы для автоматического рестарта API и Worker.

Как применять:
1. Распаковать архив в корень текущего рабочего проекта с заменой файлов.
2. Для backend выполнить:
   `cd backend && source .venv/bin/activate && pip install -e .[dev] && alembic upgrade head`
3. Для frontend выполнить в корне проекта:
   `npm install && npm run build` или `npm run dev`.

Важно:
- этот патч собран в текущей сессии поверх исходного проекта, потому что последний рабочий архив заказчика в сессию не загружен;
- изменения сделаны максимально точечно, чтобы их можно было безопасно наложить на текущую рабочую ветку.
