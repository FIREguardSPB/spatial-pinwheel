# Руководство пользователя Spatial Pinwheel (Trading Bot)

## Что это такое?

Spatial Pinwheel — это автоматический торговый бот для Московской биржи (MOEX). Он анализирует графики акций, находит выгодные моменты для покупки или продажи, и может торговать самостоятельно или подсказывать вам, когда стоит войти в сделку.

---

## Основные понятия (для новичков)

### Что такое торговый сигнал?

Сигнал — это рекомендация бота: «сейчас хороший момент купить (или продать) акцию». Сигнал содержит:

- **Инструмент** — какую акцию покупать (например, SBER = Сбербанк)
- **Направление** — BUY (покупка) или SELL (продажа)
- **Цена входа (Entry)** — по какой цене открыть сделку
- **Стоп-лосс (SL)** — цена, при которой закрыть убыточную сделку (защита от больших потерь)
- **Тейк-профит (TP)** — цена, при которой закрыть прибыльную сделку (зафиксировать прибыль)

### Что такое R/R (Risk/Reward)?

Это соотношение «сколько я могу потерять» к «сколько я могу заработать». Например, R/R = 2.0 означает: рискуя 100 рублями, можно заработать 200. Бот не открывает сделки с R/R ниже 1.5 — это защита от невыгодных входов.

### Что такое стоп-лосс?

Стоп-лосс (SL) — это «страховка». Если цена пошла против вас, сделка автоматически закрывается с маленьким убытком, не давая потерять больше запланированного. Например: купили акцию за 270 руб, стоп-лосс на 265 руб — максимальный убыток 5 руб на акцию.

### Что такое тейк-профит?

Тейк-профит (TP) — это «цель». Когда цена достигает вашей цели, сделка закрывается автоматически с прибылью. Не нужно сидеть перед экраном и ждать.

## Страницы приложения

### Дашборд (главная)

Основной экран с графиком цен и текущим состоянием:

- **График** — свечной график выбранного инструмента в реальном времени
- **Выбор инструмента** — переключение между акциями (SBER, GAZP, LKOH и другие)
- **Статус подключения** — зелёный = работает, красный = нет связи
- **Виджеты** — текущий P&L, количество открытых позиций, дневная статистика

### Сигналы

Список всех торговых сигналов, найденных ботом:

- **Статус сигнала:**
  - `pending_review` — ожидает вашего решения (одобрить или отклонить)
  - `approved` — одобрен, передан на исполнение
  - `executed` — исполнен, сделка открыта
  - `rejected` — отклонён (вами или ботом)

- **Действия:**
  - Кнопка «Одобрить» — открыть сделку по этому сигналу
  - Кнопка «Отклонить» — пропустить этот сигнал

- **Карточка AI-решения** — если AI включён, здесь видно что думает искусственный интеллект: его решение, уверенность (0–100%), и ключевые факторы

### Сделки

Журнал всех исполненных сделок с фильтрами:

- Какой инструмент, когда, в каком направлении
- Результат: прибыль или убыток
- Экспорт в CSV для анализа в Excel

### Счёт

Информация о вашем торговом счёте:

- **Баланс** — сколько денег на счёте
- **Equity** — баланс + нереализованная прибыль/убыток по открытым позициям
- **Дневной P&L** — прибыль/убыток за сегодня
- **Кривая equity** — график изменения вашего счёта во времени

### Бэктест

Тестирование стратегии на исторических данных (без риска реальными деньгами):

1. Выберите стратегию
2. Загрузите исторические данные (свечи)
3. Нажмите «Запустить»
4. Посмотрите результаты: сколько бы заработали, максимальная просадка, процент прибыльных сделок

### События

Лог всех действий бота: какие решения принимались, почему сигналы были одобрены или отклонены.

### Настройки

Здесь настраивается поведение бота.

#### Риск-менеджмент

| Параметр | Что значит | Рекомендация для новичков |
|----------|-----------|--------------------------|
| Риск на сделку (%) | Какую часть счёта рисковать в одной сделке | 0.5–1% (не больше!) |
| Дневной лимит потерь (%) | Бот остановится, если потеряете столько за день | 2–3% |
| Макс. позиций одновременно | Сколько сделок может быть открыто | 2–3 |
| Макс. сделок в день | Лимит количества сделок | 5–8 |
| Стоп по времени (баров) | Закрыть позицию если она не двигается N баров | 6 |

#### Стратегия

Три встроенные стратегии:

| Стратегия | Когда покупает | Для какого рынка |
|-----------|---------------|------------------|
| **Breakout** | Цена пробивает максимум/минимум за N баров | Трендовый (когда рынок двигается) |
| **Mean Reversion** | Цена отскакивает от края Bollinger Bands | Боковой (когда рынок стоит на месте) |
| **VWAP Bounce** | Цена отскакивает от средневзвешенной цены дня | Внутридневной |

Для начала используйте **Breakout** — она самая понятная и надёжная.

#### Режим торговли

| Режим | Что делает |
|-------|-----------|
| **Review** (рекомендуется новичкам) | Бот находит сигналы, но не торгует — ждёт вашего одобрения |
| **Auto Paper** | Бот торгует автоматически, но виртуальными деньгами |
| **Auto Live** | Бот торгует реальными деньгами (требует подключение к брокеру) |

**Совет:** Начните с Review, потом попробуйте Auto Paper на неделю, и только убедившись в результатах — переходите к Auto Live.

#### AI-советник

AI анализирует сигналы бота и даёт свою оценку:

| Режим | Что делает |
|-------|-----------|
| **Off** | AI выключен, решения принимает только Decision Engine |
| **Advisory** | AI даёт совет, но не влияет на решение (вы видите его мнение) |
| **Override** | AI может изменить решение бота, если очень уверен (> 70%) |
| **Required** | Без одобрения AI ни одна сделка не пройдёт |

Поддерживаемые AI-провайдеры: Claude (Anthropic), GPT-4o (OpenAI), локальные модели через Ollama.

### Токены

Управление API-ключами (все ключи хранятся зашифрованно):

- **Auth Token** — пароль для доступа к боту
- **Claude / OpenAI API Key** — для AI-анализа
- **Telegram Bot Token** — для уведомлений в Telegram
- **T-Bank Token** — для реальной торговли

Кнопка «Тест» проверяет работоспособность каждого ключа.

## Как начать (пошагово)

### Шаг 1: Знакомство (10 минут)

1. Запустите бот в mock-режиме (см. инструкцию по развёртыванию)
2. Откройте http://localhost:5173
3. Посмотрите на дашборд — увидите график с фиктивными ценами
4. Перейдите в «Сигналы» — увидите сигналы, которые бот нашёл
5. Нажмите на любой сигнал — изучите карточку (Entry, SL, TP, Score)

### Шаг 2: Настройка (5 минут)

1. Перейдите в «Настройки»
2. Убедитесь что режим торговли = **Review** (ручное подтверждение)
3. Стратегия = **Breakout**
4. Риск на сделку = **1%**

### Шаг 3: Попробуйте AI (необязательно)

1. Перейдите в «Токены»
2. Добавьте ключ Claude API или OpenAI
3. В «Настройках» включите AI режим = **Advisory**
4. Теперь в карточке каждого сигнала будет мнение AI

### Шаг 4: Бумажная торговля

1. В «Настройках» переключите режим на **Auto Paper**
2. Бот начнёт автоматически открывать и закрывать виртуальные сделки
3. Наблюдайте за результатами на странице «Счёт»
4. Рекомендуется наблюдать минимум 1–2 недели

### Шаг 5: Telegram-уведомления (рекомендуется)

1. Создайте бота в Telegram через @BotFather
2. Добавьте токен бота на странице «Токены»
3. Узнайте свой Chat ID через @userinfobot
4. Добавьте Chat ID на странице «Токены»
5. Теперь бот будет присылать уведомления о сигналах и сделках

## Что означают оценки бота

### Decision Engine Score (0–100)

Это комплексная оценка качества сигнала. Чем выше — тем лучше:

| Компонент | Вес | Что проверяет |
|-----------|-----|---------------|
| Тренд (Regime) | 20 | Цена выше скользящей средней? Тренд попутный? |
| Волатильность | 15 | Стоп-лосс адекватен текущей волатильности? |
| Момент (RSI + MACD) | 15 | Есть ли импульс в нужную сторону? |
| Уровни | 20 | Нет ли сильного сопротивления на пути к цели? |
| Издержки | 15 | Комиссии не съедят всю прибыль? |
| Объём | 10 | Достаточно ли объёма на рынке? |

Если score ≥ порога (по умолчанию 70) — бот рекомендует TAKE (открыть сделку).

### AI Confidence (0–100)

Уверенность AI-советника в своём решении:
- 80–100: AI очень уверен
- 60–80: средняя уверенность
- ниже 60: AI не уверен (в режиме Override не будет менять решение DE)

## Часто задаваемые вопросы

**Могу ли я потерять деньги?**
В режиме Paper — нет, это виртуальные деньги. В режиме Live — да, торговля всегда связана с риском. Бот минимизирует убытки через стоп-лоссы и лимиты, но не гарантирует прибыль.

**Какой минимальный счёт нужен?**
Для бумажной торговли — 0 руб (виртуальные деньги). Для реальной торговли на MOEX — рекомендуется от 50 000 руб.

**Бот торгует 24/7?**
Нет. Бот работает только во время торговой сессии MOEX (09:50–18:50 МСК, пн–пт). Первые 10 минут после открытия и последние 10 минут перед закрытием бот не открывает новые сделки (защита от открытия).

**Что если интернет пропадёт?**
Открытые позиции остаются. Стоп-лоссы на бирже не выставляются автоматически (это бумажная торговля) — они проверяются ботом на каждом тике. Если бот упал, позиции остаются открытыми до перезапуска.

**Можно ли торговать несколькими акциями?**
Да. Добавьте инструменты через страницу «Настройки» → раздел «Инструменты» или через API watchlist. По умолчанию отслеживаются SBER, GAZP, LKOH.

**Что такое Paper Trading?**
Бумажная (виртуальная) торговля — бот делает всё то же самое что и при реальной торговле, но без денег. Идеально для тестирования стратегий.

## Глоссарий

| Термин | Объяснение |
|--------|-----------|
| **ATR** | Average True Range — средний диапазон свечи. Мера волатильности. |
| **Bollinger Bands** | Полосы вокруг средней цены. Цена обычно движется внутри полос. |
| **BUY** | Покупка акции (ставка на рост цены) |
| **DE** | Decision Engine — движок принятия решений бота |
| **EMA** | Exponential Moving Average — скользящая средняя (индикатор тренда) |
| **Entry** | Цена входа в сделку |
| **Equity** | Баланс + нереализованная прибыль/убыток |
| **MACD** | Moving Average Convergence Divergence — индикатор импульса |
| **MOEX** | Московская биржа |
| **P&L** | Profit & Loss — прибыль и убыток |
| **R/R** | Risk/Reward — соотношение риска к потенциальной прибыли |
| **RSI** | Relative Strength Index — индикатор перекупленности/перепроданности |
| **SELL** | Продажа акции (ставка на падение цены) |
| **SL** | Stop-Loss — уровень автоматического закрытия убыточной позиции |
| **TP** | Take-Profit — уровень автоматического закрытия прибыльной позиции |
| **VWAP** | Volume Weighted Average Price — средневзвешенная цена по объёму |
