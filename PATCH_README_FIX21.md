# FIX21 — sector-aware AI context

## Что добавлено
- Введён слой `sector_profiles` для всего текущего пула бумаг из каталога watchlist/instruments.
- AI-контекст теперь отраслевой, а не «нефтяной по умолчанию».
- Расширены `ticker/sector keywords` для news collector, чтобы компании из разных отраслей получали более релевантный новостной поток.
- Prompt теперь явно сообщает сектор бумаги и её профильные драйверы.
- Narrative summary и фундаментальный блок стали sector-aware.

## Какие сектора покрыты
- нефтегаз
- банки и финансовая инфраструктура
- технологии и интернет
- телеком
- транспорт и логистика
- ритейл и e-commerce
- металлы и добыча
- электроэнергетика и сети
- девелоперы и недвижимость
- удобрения и агро
- лес и упаковка

## Что это даёт
- `YNDX/VKCO` больше не анализируются логикой Brent-first
- `AFLT` получает оценку через топливо / маршруты / санкции / лизинг
- `MGNT/OZON/FIXP` — через потребительский спрос / инфляцию / логистику / FX costs
- `MTSS/RTKM` — через CAPEX / тарифы / импорт оборудования
- `SBER/VTBR/MOEX` — через ставку / кредитный спрос / ликвидность / санкции
